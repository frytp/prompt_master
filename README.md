# Prompt Manager

Веб-приложение для управления промптами для работы с AI-моделями.

## Описание

Prompt Manager помогает организовать и хранить готовые промпты для различных AI-моделей (ChatGPT, Claude, Gemini). Приложение позволяет:

- ✅ Создавать и редактировать промпты
- ✅ Организовывать промпты по категориям, тегам и коллекциям
- ✅ Использовать шаблонные переменные `{{переменная}}`
- ✅ Фильтровать и искать промпты по множественным критериям
- ✅ Импортировать готовые наборы промптов из JSON
- ✅ Копировать промпты в буфер обмена одним кликом
- ✅ Отслеживать статистику использования
- ✅ Добавлять промпты в избранное
- ✅ Просматривать историю использованных значений переменных

## Технологии

- Python 3.10+
- Django 5.0
- Bootstrap 5
- JavaScript (AJAX)
- SQLite

## Установка и запуск

### 1. Клонировать репозиторий
```bash
git clone https://github.com/your-username/prompt-manager.git
cd prompt-manager
```

### 2. Создать виртуальное окружение
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate
```

### 3. Установить зависимости
```bash
pip install -r requirements.txt
```

### 4. Применить миграции
```bash
python manage.py makemigrations
python manage.py migrate
```

### 5. Загрузить предустановленные данные (опционально)
```bash
python manage.py loaddata prompts/fixtures/initial_data.json
```

### 6. Создать суперпользователя
```bash
python manage.py createsuperuser
```

### 7. Запустить сервер
```bash
python manage.py runserver
```

Приложение будет доступно по адресу: http://127.0.0.1:8000/

## Основные возможности

### 1. Шаблонные переменные

Используйте `{{переменная}}` в тексте промпта для создания переменных шаблонов:
```
Объясни {{тема}} простыми словами для {{аудитория}}.
```

При использовании промпта система предложит заполнить переменные и сохранит историю значений.

### 2. Фильтры и поиск

- Поиск по названию
- Множественный выбор категорий
- Множественный выбор тегов
- Фильтр по коллекциям
- Фильтр избранного
- Сортировка по популярности, дате, алфавиту

### 3. Статистика

Просмотр статистики использования:
- Топ-10 популярных промптов
- Статистика по категориям
- Статистика по тегам
- Статистика по AI моделям
- Статистика по коллекциям

### 4. Импорт промптов

Импорт промптов из JSON файла с автоматическим созданием категорий, тегов и AI моделей.

Пример структуры JSON:
```json
[
  {
    "title": "Название промпта",
    "content": "Текст промпта с {{переменными}}",
    "category": "Образование",
    "collection": "Моя коллекция",
    "tags": ["объяснение", "примеры"],
    "ai_models": ["ChatGPT", "Claude"]
  }
]
```

## Модели данных

- **Prompt** - основная модель промпта
- **Category** - категория промпта
- **Tag** - тег для классификации
- **AIModel** - AI модель (ChatGPT, Claude и т.д.)
- **Collection** - коллекция (папка) для организации промптов
- **PromptVariable** - история использованных значений переменных
